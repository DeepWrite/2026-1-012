# 과제-03 단문 연습에 대한 코멘트

- 대상과제: `과제-03 단문 연습`
- 코멘트를 제공하는 학생: `012-11 이서연` 
- 코멘트를 받는 학생: `012-23 박성윤` 

---

## 코멘트

현재 제출된 OREO 단문 연습 과제는 논리적 글쓰기를 훈련을 효과적으로 수행하였다. 논제 진술(Opinion)이 제시한 배경 및 난제와 유기적인 흐름을 가진다. 따라서, 독자가 글의 초점을 명확히 이해하고 작성자의 논증을 파악하기에 용이하다. 이와 더불어, 개인의 데이터가 관계적 외부효과를 지니기에 개인의 권리 보호만으로는 데이터 독점을 해결하기에 불충분하다는 이유(Reason)와 증거(Example)로 제시된 알고리즘의 데이터 분석 방식 사례가 논리적으로 명확히 연결되어있다. 이에 더불어, 상향식 데이터 신탁이 알고리즘의 데이터 분석 방식에 어떻게 효과적으로 대응할 수 있는지 등의 사례를 추가하면 논증에 대한 설득력이 높아질 것이다. 마지막으로, 재진술(Opinion Restated) 부분은 이전에 제시한 논거와 논증을 단순한 요약을 넘어 효과적으로 강조하고 있으며, 이전 논거를 통합적으로 제시하여 논증을 강화하고 있다.

---

## 체크 포인트

### **구성요소**
- [ ] 논제 진술문을 찾거나 식별하기 어렵다.
- [ ] 논증 진술문을 찾거나 식별하기 어렵다.
- [ ] 증거/사례 진술문을 찾거나 식별하기 어렵다.
- [ ] 논제 재-진술문을 찾거나 식별하기 어렵다.
- [x] 논제, 논증, 증거/사례, 논제 재-진술문 각각 기능적으로 명확히 구분되어 진술되었다.

### **논제 진술의 명확성 및 표현** 
- [ ] 논제 진술문이 너무 일반적이거나 모호하여 독자가 핵심 주장을 명확히 이해하기 어렵다. 
- [ ] 논제 진술문의 핵심 용어 선택이 모호하거나, 애매하거나, 잘못되었다. 
- [x] 논제 진술문이 필자의 의도를 명확하게 전달하고 있다. 

### **논증과 증거/사례 제시** 
- [ ] 이유(Reason)가 논제와 무관하다.
- [ ] 이유(Reason)가 논제를 충분히 뒷받침하지 못하고 있으며, 명확한 논리적 연결이 부족하다. 
- [ ] 증거(Example)가 구체적이지 않거나, 논제 및 이유와의 관계가 명확하지 않다. 
- [ ] 증거(Example)가 사실적 근거보다는 개인적 견해에 의존하고 있다. 
- [x] 이유와 증거가 논리적으로 명확하게 연결되었다. 

### **논제 재-진술** 
- [ ] 논제 재-진술문이 적절하지 않거나, 단순한 반복에 그친다. 
- [x] 논제 재-진술문이 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행한다.
