# 과제-03 단문 연습에 대한 코멘트

- 대상과제: `과제-03 단문 연습`
- 코멘트를 제공하는 학생: `012-23 박성윤` 
- 코멘트를 받는 학생: `012-15 손예원` 

---

## 코멘트

현재 제출된 OREO 논증문은 "공연예술의 존재 본질은 소멸에 있다"는 핵심 주장을 간결하게 드러내고 있고 "공연예술은 현장에서의 감각적 경험을 제공하는 것을 핵심으로 하는 순간의 예술이며, 실시간으로 이루어진 즉시 소멸하여 기억 속에만 존재하게 되기 때문이다."라는 이유도 논제를 잘 뒷받침하고 있다. 다만 "예를 들어, 카메라를 통해 공연을 촬영하여 이를 재생산하거나 기록할 경우 생성된 영상물은 공연의 장면을 담고 있기는 하지만 라이브 공연 순간 관객이 현장에서 직접 느낀 감각적 경험을 완전하게 제공할 수 없다."라는 증거는 주장과 이유의 흐름을 잘 보여주는 논리적 예시이기는 하지만 데이터나 사실적 근거라기보다는 개념적 부연 설명에 가깝다. 이를 개선하기 위해 증거 부분에서 다른 학자의 연구에서 이론적 견해를 인용하거나 구체적인 역사적 사례나 비평 사례를 제시하는 등의 방식으로 수정하는 것이 필요하다.

## 체크 포인트

### **구성요소**
- [ ] 논제 진술문을 찾거나 식별하기 어렵다.
- [ ] 논증 진술문을 찾거나 식별하기 어렵다.
- [ ] 증거/사례 진술문을 찾거나 식별하기 어렵다.
- [ ] 논제 재-진술문을 찾거나 식별하기 어렵다.
- [x] 논제, 논증, 증거/사례, 논제 재-진술문 각각 기능적으로 명확히 구분되어 진술되었다.

### **논제 진술의 명확성 및 표현** 
- [ ] 논제 진술문이 너무 일반적이거나 모호하여 독자가 핵심 주장을 명확히 이해하기 어렵다. 
- [ ] 논제 진술문의 핵심 용어 선택이 모호하거나, 애매하거나, 잘못되었다. 
- [x] 논제 진술문이 필자의 의도를 명확하게 전달하고 있다. 

### **논증과 증거/사례 제시** 
- [ ] 이유(Reason)가 논제와 무관하다.
- [ ] 이유(Reason)가 논제를 충분히 뒷받침하지 못하고 있으며, 명확한 논리적 연결이 부족하다. 
- [ ] 증거(Example)가 구체적이지 않거나, 논제 및 이유와의 관계가 명확하지 않다. 
- [x] 증거(Example)가 사실적 근거보다는 개인적 견해에 의존하고 있다. 
- [x] 이유와 증거가 논리적으로 명확하게 연결되었다. 

### **논제 재-진술** 
- [ ] 논제 재-진술문이 적절하지 않거나, 단순한 반복에 그친다. 
- [x] 논제 재-진술문이 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행한다.
