# 과제-03 단문 연습에 대한 코멘트

- 대상과제: `과제-03 단문 연습`
- 코멘트를 제공하는 학생: `012-10 조성우` 
- 코멘트를 받는 학생: `012-14 정현노` 

---

## 코멘트

해당 단문은 로크의 전유 이론에 대하여 O-R-E-O의 형식으로 논증을 구성하였다. 논증에 대한 근거, 논제 재-진술은 명확히 되었으나, 처음 Opinion에 있어서 다소 의문이 드는 지점이 있다. 논제 진술문에서는 어떤 식으로든 작성자의 주장이 드러나야하는데, 해당 단문에서는 주장보다는 로크의 이론에 대한 문제 제기에 가까운 듯한 느낌이다. 물론 문제 제기도 큰 범주에서는 주장이라고 볼 수도 있으나 작성자 본인의 주장이라는 점이 더욱 잘 드러나면 좋을 것 같다. 또, 예시의 적절성도 다시 생각해볼 필요가 있어 보인다. 로크의 전제 조건을 위반하게 되는 예시는 결국 로크의 전유 이론과 화폐에 대한 주장에 있어서 어긋나는 예시이다. 다시 말해 로크의 이론에 부합하는 사례가 아니기 때문에 로크의 주장에 대한 문제제기로부터 시작한 해당 논증에 알맞은 예시인지가 다소 의문이다. 로크의 전제를 위반하지 않으면서 자연법적 제한을 무력화하는 사례를 드는 것이 해당 논증에 더 알맞은 예시일 것이라 사료된다.

## 체크 포인트

### **구성요소**
- [x] 논제 진술문을 찾거나 식별하기 어렵다.
- [ ] 논증 진술문을 찾거나 식별하기 어렵다.
- [ ] 증거/사례 진술문을 찾거나 식별하기 어렵다.
- [ ] 논제 재-진술문을 찾거나 식별하기 어렵다.
- [ ] 논제, 논증, 증거/사례, 논제 재-진술문 각각 기능적으로 명확히 구분되어 진술되었다.

### **논제 진술의 명확성 및 표현** 
- [ ] 논제 진술문이 너무 일반적이거나 모호하여 독자가 핵심 주장을 명확히 이해하기 어렵다. 
- [x] 논제 진술문의 핵심 용어 선택이 모호하거나, 애매하거나, 잘못되었다. 
- [ ] 논제 진술문이 필자의 의도를 명확하게 전달하고 있다. 

### **논증과 증거/사례 제시** 
- [ ] 이유(Reason)가 논제와 무관하다.
- [ ] 이유(Reason)가 논제를 충분히 뒷받침하지 못하고 있으며, 명확한 논리적 연결이 부족하다. 
- [x] 증거(Example)가 구체적이지 않거나, 논제 및 이유와의 관계가 명확하지 않다. 
- [ ] 증거(Example)가 사실적 근거보다는 개인적 견해에 의존하고 있다. 
- [ ] 이유와 증거가 논리적으로 명확하게 연결되었다. 

### **논제 재-진술** 
- [ ] 논제 재-진술문이 적절하지 않거나, 단순한 반복에 그친다. 
- [ ] 논제 재-진술문이 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행한다.
