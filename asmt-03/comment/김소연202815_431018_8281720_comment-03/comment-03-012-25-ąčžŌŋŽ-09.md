# 과제-03 단문 연습에 대한 코멘트

- 코멘트를 받는 학생: `012-09 황원이` 

---

## 코멘트
 
현재 제출된 OREO 논증문은 논제 진술이 명확하게 서술되어 있어 식별하기 쉬웠다. 또한 마지막에 다시한번 본인의 주장을 재진술할 때 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행하고 있어 주장을 강조하는 효과가 있었다.
그러나,사례에서 제시된 경제적 상황이 좋지 않은 개인의 경우 정치에 지속적으로 참여하기 보다 생계를 유지하는데에 더 집중할 가능성이 크다고 서술된 부분에서 구체적인 증거를 찾을 수 없으며 실제 통계라던지 관련된 문헌을 인용하여 서술하였다면 본인의 주장에 대한 근거를 구체적으로 확인할 수 있을 것 같다. 본 주제자체가 추상적인 부분이다 보니 글 전체가 모호하게 느껴질 수 있기 때문에  '정치적 평등을 완전히 부정하기보다 정치적으로 소외된 다수의 이해를 반영할 수 있는 장치를 활용하여 실질적인 정치적 평등을 추구하는 것이 옳다'는 부분에 구체적으로 어떤 장치를 의미하는 것인지 제시한다면 주장의 명확성을 높일 수 있을 것처럼 보인다.

## 체크 포인트

### **구성요소**
- [ ] 논제 진술문을 찾거나 식별하기 어렵다.
- [ ] 논증 진술문을 찾거나 식별하기 어렵다.
- [ ] 증거/사례 진술문을 찾거나 식별하기 어렵다.
- [ ] 논제 재-진술문을 찾거나 식별하기 어렵다.
- [ ] 논제, 논증, 증거/사례, 논제 재-진술문 각각 기능적으로 명확히 구분되어 진술되었다.

### **논제 진술의 명확성 및 표현** 
- [x] 논제 진술문이 너무 일반적이거나 모호하여 독자가 핵심 주장을 명확히 이해하기 어렵다. 
- [x] 논제 진술문의 핵심 용어 선택이 모호하거나, 애매하거나, 잘못되었다. 
- [ ] 논제 진술문이 필자의 의도를 명확하게 전달하고 있다. 

### **논증과 증거/사례 제시** 
- [ ] 이유(Reason)가 논제와 무관하다.
- [ ] 이유(Reason)가 논제를 충분히 뒷받침하지 못하고 있으며, 명확한 논리적 연결이 부족하다. 
- [X] 증거(Example)가 구체적이지 않거나, 논제 및 이유와의 관계가 명확하지 않다. 
- [x] 증거(Example)가 사실적 근거보다는 개인적 견해에 의존하고 있다. 
- [ ] 이유와 증거가 논리적으로 명확하게 연결되었다. 

### **논제 재-진술** 
- [ ] 논제 재-진술문이 적절하지 않거나, 단순한 반복에 그친다. 
- [X] 논제 재-진술문이 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행한다.
