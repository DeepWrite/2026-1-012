# 과제-07 개인별 논증 구조 작성하기 012-20 육민서

## 제목: LLM 서비스 제공자의 원자료에 대한 규범적 보상 의무 

## 1. 쟁점과 딜레마

| 구분 | 내용 |
|:---|:---|
| 주제(Topic) | LLM 서비스 제공자의 원자료 생산자에 대한 규범적 보상의무 |
| 도전하려는 쟁점 | LLM 답변의 효용 형성에 원자료의 정보가 결정적으로 기여하는 경우, 사용자가 LLM 서비스에 지불한 대가를 원자료 생산자와 나눠야 하는가? |
| 딜레마/난제 | LLM 서비스 이용 대가의 귀속은 서비스 효용의 기여 주체를 기준으로 하는가, 아니면 사용자의 지불 의도를 기준으로 하는가? |
| 딜레마/난제 해소/해결 방법 | 서비스에 지불하는 대가는 서비스가 제공한 효용에 대한 것이며, LLM 서비스의 효용은 원자료의 정보를 포함한다. |

① 주제(Topic): LLM 답변이 원자료의 정보 제공 기능을 재현할 때, LLM 서비스 제공자의 원자료 생산자에 대한 규범적 보상 의무

② 도전하는 학술적 쟁점:LLM 답변이 원자료의 정보 제공 기능을 재현할 때, 원자료 생산자는 기여분에 따른 규범적 청구권을 LLM 서비스 제공자에게 행사할 수 있는가?

- **원자료의 정보가 LLM 답변의 정보적 속성과 정보제공 효용에 결정적으로 기여하는가?** 
- **원자료가 LLM 답변의 효용에 기여를 한다는 전제가 원자료 생산자에게 규범적 청구권을 부여하는가?** 
- **LLM 서비스 이용자가 대가를 지불할 때, 원자료의 정보 제공 효용이 아닌 LLM의 정보 처리 능력에 대해서만 지불할 의도를 지녔더라도, 원자료 생산자는 LLM 서비스 제공자에게 보상을 받을 권리가 있는가?**

③ 유발되는 딜레마 또는 난제

- 딜레마 구조
 - **(A)** LLM 서비스 이용 대가는 LLM 서비스의 효용에 대한 것이며, 원자료의 정보는 LLM 서비스의 효용에 결정적으로에기여하기 때문에, LLM 서비스 이용 대가는 원자료의 몫을 포함한다.
 - **(B)** LLM 서비스의 사용자가 원자료의 정보가 아니라, 오직 LLM 서비스의 정보 처리 능력을 위해 대가를 지불한다면, LLM 서비스 이용 대가에는 원자료 생산자의 몫이 포함되지 않는다.

④ 딜레마 해소 (또는 난제 해결) 전략

- 원자료의 정보 제공 기능은 LLM 답변의 효용을 구성하는 결정적 요소이다. (Woodruff, 1997)
- 효용에 결정적으로 기여한 주체는 기여분에 상응하는 규범적 청구권을 지닌다.(Lamont, 1994)
- 기여분에 따른 청구권은 기여분이 포함된 대가를 수취하거나 보유한 주체에게 향한다. (Cullity, 1995)
- LLM 서비스 제공자는 원자료의 정보 제공기능이 포함된 효용의 대가를 일괄 수취하면서 원자료 생산자의 기여분을 별도로 귀속시키지 않으므로, 원자료 생산자에 대한 보상의무를 진다.

## 2. 논증구조

### 기본구조

- **논제:** LLM 답변이 특정 원자료의 정보 제공 기능을 재현할 때, LLM 서비스 제공자에게 원자료 생산자의 기여분에 상응하는 보상 의무가 발생한다.
 - **전제1:** LLM 답변이 원자료의 정보 제공 기능을 재현하는 경우, 원자료의 정보 제공 기능은 LLM 답변의 정보제공 효용에 결정적으로 기여한다.
  - 산출물의 효용은 산출물 자체에서 발생하는 것이 아니라, 1)산출물의 속성, 2)그 속성이 사용 상황에서 낳는 결과, 3)그 결과가 사용자의 목표와 목적을 촉진하는 일련의 과정에서 형성된다. (Woodruff, 1997)
  - LLM 답변의 효용도 답변 자체에서 나오는 것이 아닌, 답변이 제공하는 정보적 속성, 그 속성이 낳는 이해, 판단, 비교, 선택 등의 사용 결과, 그리고 그 결과가 사용자의 목표 달성을 촉진하는 과정에서 발생한다.
  - LLM 답변이 특정 원자료의 정보 제공 기능을 재현할 때, 그 원자료의 정보는 답변의 정보적 속성을 형성하며, 그 결과 답변의 정보제공 효용에 결정적으로 기여한다.
 - **전제2:** 효용에 결정적으로 기여한 주체는 그 기여분에 대한 청구권을 가진다. 그 청구권은 해당 기여분이 포함된 대가를 수취하거나 보유한 주체에게 향한다.
  - 서비스의 대가는 그 서비스가 제공하는 효용에 대해 지급된다.
  - 서비스 효용이 복수 주체의 기여로 형성되면 그 대가에는 각 기여에 상응하는 몫이 포함된다.
  - 따라서 효용에 결정적으로 기여한 주체는 그 기여분에 상응하는 몫에 대한 청구권을 지닌다. (Lamont, 1994)
  - 이 청구권은 그 기여분이 포함된 대가를 수취하거나 보유한 주체에게 향한다. (Cullity, 1995)
 - **전제3:** LLM 서비스 제공자는 LLM의 정보처리능력과 원자료의 정보제공기능이 결합된 효용의 대가를 일괄 수취하면서 원자료 생산자의 기여분을 별도로 귀속시키지 않는다.
  - LLM 서비스 제공자는 구독료, 광고 등의 형태로 LLM 답변 효용에 대한 대가를 수취한다.
  - 그 대가는 LLM 자체의 정보 처리 능력과 원자료의 정보 제공 기능이 결합된 효용에 대한 것이다.
  - 원자료 생산자의 기여분은 별도로 귀속되지 않는다.
  - LLM 서비스 제공자는 원자료 생산자의 기여분이 포함된 효용의 대가를 일괄 수취한다. 즉, LLM 서비스 제공자는 원자료 생산자의 기여몫을 내부화한다.

- **결론:** 따라서, LLM 답변이 특정 원자료의 정보 제공 기능을 재현할 때, LLM 서비스 제공자에게 원자료 생산자의 기여분에 상응하는 보상 의무가 발생한다.

### 예상반론과 재반박

- **예상반론(연역적 논증의 타당성 공격):** **전제2**와 **전제3**에서 서비스의 대가가 효용에 대해 지급된다는 주장은 대가를 지급하는 이의 의도를 반영하지 않는다.
 - 논리적 취약점 지적: 사용자는 원자료의 정보 자체를 위해 LLM 서비스에 대가를 지불하는 것이 아니다. 원자료의 정보는 포털을 검색하거나 원문을 통해 얻을 수 있다. 추가적인 대가를 치르는 이유는 LLM의 정보 처리 기능 때문이다. 원자료가 LLM의 답변에 기여할 수는 있지만, 사용자가 대가를 지불한 대상은 원자료가 아닌 LLM의 정보 처리 기능이다.

- **재반박:** 사용자가 대가를 지불한 주관적 의도와, 그 대가의 정당한 귀속 대상은 구분되어야 한다. 독자는 외국어 원서를 읽는 대신 한국어 번역본을 더 비싼 값에 구매한다. 구매의 의도는 번역자가 익숙한 언어로 내용을 변환해주기 때문이지만, 지불하는 대가는 번역과 그 책의 내용 자체를 모두 포함한다. 번역본의 예시와 같이, 주관적 의도와 대가의 기여분 포함 여부는 별개의 것이다.
